# Plan — Template Premium Agence Marketing & Communication (Secteur 05)

## Overview
Build a production-ready, premium React website template for a Marketing & Communications Agency. The site must demonstrate creative excellence and measurable results — it IS the agency's first campaign.

## Skill Loading
- **Stage 1-2**: Orchestrator designs design.md and content.md directly (skill self-design)
- **Stage 3-5**: Load `vibecoding-webapp-swarm` for development

## Stage 1: Design System
- Create design.md with:
  - Color palette: Creative & energetic (rose #FF006E, jaune #FFBE0B, bleu #3A86FF) on dark/light
  - Typography: Geist Sans + Geist Mono per skill spec
  - Spacing, layout grid, animation patterns
  - Component library specs

## Stage 2: Content Planning
- Create content.md with all page content:
  - 7 public pages: Home, Campagnes, Services, Clients, Équipe, Insights, Contact
  - Realistic demo content for a fictional agency "NOVA"
  - All text in French

## Stage 3: Asset Generation
- Generate all required images:
  - Hero campaign visuals
  - Portfolio campaign images (6-8 campaigns)
  - Team portraits (6 members)
  - Client logos (placeholder style)
  - Blog article covers

## Stage 4: Development
- Load `vibecoding-webapp-swarm` skill
- Build the complete Next.js + Tailwind + shadcn/ui template
- All 7 pages with responsive design
- Smooth scroll animations, dark/light mode
- Deploy

## Stage 5: Polish & Deploy
- SEO metadata & schema markup
- Accessibility checks
- Deploy to production

## Pages
1. **/ (Home)** — Hero créatif + services + campagnes à la une + chiffres clés + logos clients + témoignages + CTA
2. **/campagnes** — Portfolio filtrable avec études de cas détaillées
3. **/services** — Offres par type, par canal, par secteur
4. **/clients** — Études de cas + témoignages + logos
5. **/equipe** — Membres avec spécialités
6. **/insights** — Blog, tendances, rapports
7. **/contact** — Formulaire avec type de besoin + budget
